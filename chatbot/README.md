# Rule-Based Chatbot with NLTK & Streamlit

A conversational AI chatbot that uses rule-based intent classification with NLTK preprocessing and TF-IDF cosine similarity matching.

## Features

- Text preprocessing: tokenization, lemmatization, stopword removal
- Intent classification using TF-IDF vectorization and cosine similarity
- Multiple response variations for each intent
- Interactive Streamlit web interface
- Fallback responses for unknown queries
